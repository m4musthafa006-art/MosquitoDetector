Mosquito Detector

Streamlit app using YOLO for mosquito detection.

Quick deploy (Streamlit Cloud):
1. Ensure best.pt is NOT committed. Upload model to a release or external storage.
2. Push repo to GitHub (main branch).
3. On Streamlit Cloud, create a new app from this repo, main branch, and set main file to app_webcam.py.

Files: app_webcam.py, app.py, Dockerfile, requirements.txt
